== Description ==

A WordPress plugin that adds a Slide In area containing recent posts and a Call to Action.

**In the plugin settings, you can also add Custom CSS to change the layout of the slide in displayed.**

**Please rate if you like it :)**

== Installation ==
Install the plugin and activate. Go to Recent Posts Slide-In and CTA options page under Settings menu and customize.
